# Backend 

Node Express was used for REST services and MongoDB for database.

## Gitpod Development Configuration: 
Access gitpod.io#https://github.com/Malinoski/devradar

Configure:
```
cd backend
yarn add express
yarn add nodemon -D
```

## Execute the server:

```
yarn nodemon src/index.js 
```
